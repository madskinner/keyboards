This package supports all the languages of Senegal, The Gambia and Guinea-Bissau; containing layouts for:

  • Qwerty Touch keyboards on Android and iOS
  • Qwerty (en-US) keyboards on Web, Linux and Mac
  • A Windows keyboard that auto configures it self for the following system layouts:
    qwerty(en-US), qwerty(en-UK), azerty(fr-FR), qwerty(pt-PT), qwerty(pt-BR), qwertz(de-DE), qwertz(de-CH,fr-CH), qwerty(es-ES), qwerty(it-IT)<, qwerty(nl-NL), qwerty(nn-NO,nb-NO) and qwerty(sv-SW).
    The Windows keyboard will default to qwerty(en-US) if it doen't recognise your system layout.
